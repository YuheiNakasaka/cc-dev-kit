# 設計レビュー起動

design-reviewer SubAgentを使用して、設計方針をレビューしてください。

## 対象
$ARGUMENTS

引数が指定されていない場合は、`.claude/tmp/design/`配下の最新の設計書をレビューしてください。

## 実行手順

1. 設計書を読み込む
2. レビュー観点に基づいて評価
3. 改善提案を作成
4. 承認/差し戻し判定

## 出力

レビュー結果を報告し、次のアクション（実装開始または再設計）を提案してください。
