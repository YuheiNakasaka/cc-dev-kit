# 一時ファイルルール

## 保存先
作業中に作成した一時的なファイルや設計書は、必ず `.claude/tmp/` ディレクトリに作成すること。

## 対象ファイル
- 設計書・設計メモ
- 調査結果のまとめ
- スクリーンショット
- 一時的なスクリプト
- デバッグ用のログファイル
- その他の作業用ファイル

## ファイル命名規則
```
.claude/tmp/
├── design/           # 設計関連
│   └── feature-x-design.md
├── screenshots/      # スクリーンショット
│   └── ui-check-2024-01-01.png
├── research/         # 調査メモ
│   └── api-investigation.md
└── scripts/          # 一時スクリプト
    └── data-migration.rb
```

## 注意事項
- `.claude/tmp/` は `.gitignore` に設定されており、git管理されない
- 重要な情報は適切な場所に移動またはドキュメント化すること
- 定期的に不要なファイルを削除すること
